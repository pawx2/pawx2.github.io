求最大公因数的三种方法

前置知识🧀

若学习过数论/《离散数学》课程，可以很容易理解算法的正确性证明，若未学习过，下面列出部分说明帮助理解：

a | b，表示“a可以整除b”
gcd(a, b)，表示“数字a、b的最大公因数”
任意两数a、b可以相互表示 a == kb + c，简单线性函数构造

辗转相除法

正确性证明：

证明：
设有数字a，b且b > a，则可以表示a、b间关系为 b = ka + c (k、c为常数)
并且假设有 d | a 并且 d | b，则因为 c == b - ka 易推出 d | c (即 d | (b - ka))
而这里 d 可以为任何 a、b 共有因数的集合，则也包含了两者的最大公因数
因而有 gcd(a, b) = gcd(a, b % c)，并且当 a == b 时可以得到最大公因数

更相减损术

正确性证明：

证明：
设有数字a，b并且b > a
并且假设有 d | a 并且 d | b，那么显然可以推出 d | (b - a)
而这里的 d 可以表示 a、b两数的任意共有因子，也包含了两者的最大公因数
从而我们可以推出 a 与 b - a 拥有相同的因子集合
并且当 a == b 时，此时的 a 即为最大公因数

Stein算法(辗转相除法+更相减损术+位移运算)

之前还真不知道有这么个算法，开眼界了属于是🤡

正确性证明：

其正确性依赖于 辗转相除法 与 更相减损术 此处不再证明

算法原理：

移位运算性能快，并且结合以下规律可以提出Stein算法
1.a、b均为偶数，则有gcd(a, b) == 2 * gcd(a/2, b/2) == 2 * gcd(a >> 1, b >> 1);
2.a为偶数，b为奇数，则有gcd(a, b) == gcd(a/2, b) == gcd(a >> 1, b);
3.a为奇数，b为偶数，则有gcd(a, b) == gcd(a, b/2) == gcd(a, b >> 1);
4.当a、b均为奇数时，利用更相减损术，则有gcd(a, b) == gcd(b, a-b)，a-b必然为偶数，又转化为上述情况
