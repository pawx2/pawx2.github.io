LEC-ADA-EX-Chapter20 '问题与归约'习题解答

Posted on 2022-06-06 In LEC-ADA-EX
《算法设计与分析》黄宇 第二版 Chapter20 '问题与归约'习题解答，来自NJU算法设计与分析课程(LADA, Lectures of Algorithm Design And Analysis)

20.1

  • CLIQUE
    • 优化问题：输入无向图G，输出图G最大团的大小；
      判定问题：输入无向图G及常数k，判断图G中是否存在大小为k的团，若存在则返回True，否则返回False；
      1. 证明优化问题O(poly(n))可解=>证明判定问题O(poly(n))可解，
        由优化问题Alg求出Kmax，而后用k与Kmax比较，即可多项式时间内解决判定问题；
      2. 证明判定问题O(poly(n))可解=>证明优化问题O(poly(n))可解，
        调用至多n次判定问题Alg，k：1->n，当遇到返回False时，则k-1即为最大团大小，可在多项式时间内解决优化问题；
      3. 做两个方向的归约，并从形式、意义、多项式时间解释该多项式归约；
    • 构造算法Alg，其输入无向图G、常数k，随机生成一个大小为k的顶点集合，而后遍历这k个点，考察其与在集合中其它点相连的状况，如果所有点都与其它k-1个点相连，则返回True，否则返回False。
      时间复杂度为T(n) = O(n)[生成随机解] + O(n^2)[验证随机解正确性]，多项式时间可验证，为NP问题。
  • KNAPSACK
    • 优化问题：输入n个物品的重量、价值及背包容量，输出背包能容纳的最大价值；
      判定问题：输入n个物品的重量、价值及背包容量、目标价值，判断是否存在物品组合使得背包能够容纳不小于目标价值的物品，若存在则返回True，否则返回False；
      1. 证明优化问题O(poly(n))可解=>证明判定问题O(poly(n))可解，
        同CLIQUE
      2. 证明判定问题O(poly(n))可解=>证明优化问题O(poly(n))可解，
        调用至多k次，k:1->Sums(所有物品价值之和)，每次传入目标价值为k，若返回False，则k-1为优化问题解
      3. 做两个方向的归约，并从形式、意义、多项式时间解释该多项式归约；
    • 构造算法Alg，其输入n个物品的重量、价值及背包容量、目标价值下界，而后遍历n个物品，随机生成一个具有k个物品的解的集合，而后考察这k个物品的重量之和是否不大于背包容量，这k个物品的价值之和是否不小于目标价值下界。时间复杂度T(n) = O(n) + O(n) = O(poly(n))，多项式时间可验证，为NP问题。
  • INDEPENDENT-SET
    • 优化问题：输入无向图G，输出图G的最大独立集的大小；
      判定问题：输入无向图G，参数k，判断无向图G中是否存在大小为k的独立集，存在返回True，否则返回False；
    • 基本同CLIQUE问题一致
    • 基本同CLIQUE问题一致，生成随机解，而后遍历解中所有点，如果所有点的度都为0，则返回True，否则返回False。时间复杂度为T(n) = O(n) + O(n) = O(poly(n))，多项式时间可验证，为NP问题。
  • VERTEX-COVER
    • 优化问题：输入无向图G，输出图G的最小点覆盖的大小；
      判定问题：输入无向图G，参数k，判断无向图G中是否存在大小为k的点覆盖，存在返回True，否则返回False；
    • 基本同CLIQUE问题一致
    • 基本同CLIQUE问题一致，生成随机解，而后遍历解中所有点，并将关联边存入集合，最后判断集合大小是否与图G的边数相等，相等返回True，否则返回False。时间复杂度为T(n) = O(n) + O(n^2) = O(poly(n))，多项式时间可验证，为NP问题。

20.2

设P <=p Q，Q <=p R，则从形式、意义、多项式时间解释推导有P <=p R，从而证明多项式归约的传递性：

  • 形式上：构造Alg，Input_P -> T_PQ(Input_Q) -> T_QR(T_PQ(Input_Q)) -> AlgR()；
  • 意义上：说明P为YES iff R为YES，可以通过已有的P <=p Q, Q <=p R逐步说明；
  • 多项式时间上：考虑到T_PQ、T_QR均为O(poly(n))，借助多项式运算封闭性可知T_PR也为O(poly(n))；

20.3

T(n) = O(n^2) + O(n^4) = O(n^4) (归约问题 + 解决问题)

20.4

  • 选择 <=p 排序：先排序，在从已有序列进行选择
  • 排序 <=p 选择：利用选择Alg构造排序Alg，循环选出剩余数组中最大元放到结果数组末尾指示位置，并且末尾指示向前移动；

并分别从形式、意义、多项式时间说明归约的建立，令人想起选择排序

20.5

  • 问题1 <=p 问题2：循环调用问题2Alg进行排序，排序后返回中间元素
  • 问题2 <=p 问题1：利用问题1Alg选出pivot而后进行PARTITION，实现升序快排，最后返回第k个元素

并分别从形式、意义、多项式时间说明归约的建立

20.6

一个nxn方阵平方问题Alg01 <=p 两个nxn方阵相乘问题Alg02。直接调用Alg02，将Alg02的两个参数均设置为传给Alg01的参数，可通过Alg02解决Alg01。
分别从形式、意义、多项式时间说明归约的建立

20.7

  • FES问题：设计一个Alg，其随机选取图G中的边构成集合，而后从该集合中逐次删除边，并在每次删除后使用DFS检测图中是否有环，若边未删完则图无环或者边删完了仍旧有环，则返回False，否则返回True。并分析时间复杂度，说明其可在多项式时间内验证，因此属于NP问题；
